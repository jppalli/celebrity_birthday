Objective

How to Play

  2. Unscramble the letters to form a real word
  3. Type your answer in the input field
  4. The last scramble is always the author's name
  5. Click "Check Solution" when you're ready

  • Think about common words that fit the quote context

  • All letters in the scramble must be used exactly once
